# Falsy values in Python

These values are evaluated to 'false' that's why we call it `falsy` values.

```python
value_01 : None = None
value_02 : int = 0
value_03 : float = 0.0
value_04 : str = ""
Value_05 : bool = False
value_06 : list[None] = []
```
And all other empty collections and sequences are also `falsy`.
